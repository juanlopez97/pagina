Font Family

Esta propiedad nos permite cambiar la fuente de nuestro sitio. Se recomienda usar fuentes aprobadas para la web como las encontradas en este enlace: CSS Font Stack aunque tambien pueden usar las fuentes de Google: Google Fonts.

Font Size

Esta propiedad nos permite modificar el tamaño de nuestra fuente para que se vea más grande o más pequeña.

Font Weight

Esta propiedad nos permite modificar que tan "gorda" o "flaca" se ve la tipografia.

Text Align

Por defecto los textos siempre estarán alineados a la izquierda, por lo tanto, la propiedad text-align nos permite alinear el texto a la izquierda, derecha, centro y justificar.

Color y Background Color

Para modificar el color de un elemento tenemos dos propiedades color y background-color. La diferencia entre una y otra es basicamente que con la propiedad color se esta modificando el color del elemento perse en cambio cuando se modifica el background-color, lo que estamos modificando es el fondo del elemento.

Imagen de fondo y Opacidad

La opacidad es una medida que nos sirve para saber que tan transparente es un elemento. Esta se mide entre 0 y 1, en donde el 0 representa el 0% y el 1 el 100% de visibilidad.
La opacidad nos sirve para hacer un fade entre objetos y asi crear un efecto de superposición.
